# CodeGraph Support

This directory documents the optional bounded CodeGraph integration for `agent-code-router-kit`.

- `evidence-contract.md`: normalized output contract returned by the gateway
- `proof-boundaries.md`: graph discovery limits and required follow-up proof layers
- `host-integration.md`: reviewable MCP examples for supported hosts
- `benchmark-methodology.md`: separate `codegraph-router-v1` study scope

CodeGraph is off by default. The existing router remains the default path unless the user explicitly opts in.
